# 🍷 Wine Quality Prediction

A Machine Learning project that predicts the quality of wine based on its physicochemical properties. This project demonstrates the complete data analysis and machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

---

## 📌 Project Overview

Wine quality is influenced by several chemical properties such as acidity, sugar content, alcohol percentage, and pH level. This project uses machine learning algorithms to analyze these features and predict the quality of wine.

The objective is to build an accurate predictive model that can assist wine producers in assessing wine quality efficiently.

---

## 🎯 Objectives

- Analyze wine quality data
- Perform data cleaning and preprocessing
- Explore feature relationships using visualizations
- Train and evaluate machine learning models
- Predict wine quality based on input features

---

## 📂 Dataset

The dataset contains various physicochemical properties of wine samples along with their quality ratings.

### Features

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol

### Target

- Wine Quality

## ▶️ Installation

Clone the repository:

```bash

```

Navigate to the project folder:

```bash
cd Wine-Project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook:

```bash
jupyter notebook
```

Open `Wine.ipynb` and execute the cells.
